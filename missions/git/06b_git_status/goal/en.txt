Mission goal
============

Someone put (again) some  mess in this Factory.
Can you clean it?

Note that this time some change have already been put in the index !!
So you did "git add" on a file and finally do not want to keep those changes.


Useful commands
===============

git status
~~ check the status of the current git repository

git restore FILENAME
~~  restore file to status present in the index

git restore --staged FILENAME
~~  restore the file in the index to the status of the last committed version of the file
