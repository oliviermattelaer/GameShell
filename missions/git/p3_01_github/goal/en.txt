Mission goal
============

Time to go public and open source and to put the source code of the Factory online.

So first
1) create an account on github.com:
https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home

2) Go on the following web page:
https://github.com/oliviermattelaer/gitlectures
and click on "fork" (button on top right of the page)

3) click on the green button and copy the link of your repo
and then link it locally with a command like 
git clone https://github.com/olivier-cism/gitlectures.git

Useful commands
===============

git clone PATH
~~ create a local copy of a remote repository
