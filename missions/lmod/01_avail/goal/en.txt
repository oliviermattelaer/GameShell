Goal of the  mission
====================  

The command "python" is not provided by the system by default.
"python" is a very convienient scripting language which can be activated.
For your first mission, you will need to ask the librarian what is the default 
"python" version he can provide.

Then when you have the information, please run "gsh check" and tell me which version it is.

Useful commands
===============

module av XXXXX
  list various currently available program containing name XXXXX
  a (D) indicates the default version.


module load XXXXX
   activating a given program (such that you can use the related commands)

python --version
   print the version of python
